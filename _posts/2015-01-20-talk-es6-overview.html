Last night at my city's JS meetup StrasbourgJS, I talked about the new features and syntax changes of the next version of Javascript, ES6. The slides can be found here.

Luke Hoban's overview of ES6 was my main source for preparing this talk, so kudos to him for his work. If you're interested in what direction the language is taking, you should definitely read it.

If you're eager to use these features today, node.js and its fork io.js both have some level of baked-in support for them. For browser development, 6to5 is a project compiling ES6 in ES5, allowing you to use new features without worrying about browser implementation.

And if you're using Ember CLI, though it came with ES6 modules support for quite some time, the latest version (0.1.7) bundles 6to5, giving ember developers full support for ES6 features.
